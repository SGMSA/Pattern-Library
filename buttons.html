Buttons

Component

Usage


  1. This is a standard button used for accepting.
  2. It should be #339933 in colour.

HTML

<button class="main-btn">
<p class="btn-text">Button</p>
</button>

CSS

.main-btn { background: #339933;
border-radius: 5px;
height: 30px;
width: 100px;

.btn-text {
color: #fff;
}
}

Component

Usage


  1. This is the standard button for declining.
  2. The border and text of this are #339933.

HTML

<button class="hollow-btn">
<p class="btn-text-alt">Button</p>
</button>

CSS

.hollow-btn { border-radius: 5px;
height: 30px;
width: 100px;
border: 1px solid #339933;
background: #fff;

.btn-text-alt {
color: #339933;
}
}

Component

Usage


  1. This is an action button.
  2. This features within the Jumbotron.

HTML

<button class="icon-btn">
<p class="btn-text-left">Button</p>
</button>

CSS

.icon-btn {
background: #339933;
border-radius: 5px;
height: 30px;
width: 100px;

.btn-text-left {
float: left;
color: #fff;
margin-left: 10px;
}
}

.icon {
height: 15px;
width: 10px;
float: right;
margin-right: 10px;
}

Component

Usage


  1. Buttons should be aligned horizontally.
  2. They should have a margin of 15px.

HTML

<button class="btn-group-1">
<p class="btn-text">Button</p>
</button>
<button class="btn-group-2">
<p class="btn-text-alt">Button</p>
</button>

CSS

.btn-group-1 {
background: #339933;
border-radius: 5px;
height: 30px;
width: 100px;
}

.btn-group-2 {
border-radius: 5px;
height: 30px;
width: 100px;
border: 1px solid #339933;
background: #fff;
}

Component

Cancel

Usage


  1. Used as a cancelling method.
  2. Should always be underlined.

HTML

<a href="#" class="cnl">
<p>Cancel</p>
</a>

CSS

.cnl {
text-decoration: underline;
}

Component

Usage


  1. Used to confirm any large action such as payments.
  2. Highlighted in bold to show difference between other buttons.

HTML

<button class="cfm-btn">
<p class="btn-text-lrg">Button</p>
</button>

CSS

.cfm-btn {
background: #F2BC11;
border-radius: 10px;
height: 50px;
width: 150px;

.btn-text-lrg {
color: #fff;
}
}
