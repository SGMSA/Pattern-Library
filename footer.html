Footer

Component

About | Contact
2018

Usage


  1. All items should be centered and link to relative social media.
  2. Any extra meta data such as the date should feature within this area.

HTML

<div class="footer">
<img src="social.png" class="social" alt="social media">
<p class="foot-text">About | Contact <br> 2018 </p>
</div>

CSS

.footer {
border-top: 2px solid #339933;
height: 80px;
}

.social {
display: block;
margin-left: auto;
margin-right: auto;
margin-top: 10px;
}

.foot-text {
text-align: center;
margin-top: 10px;
}
